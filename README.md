# CS50X Problem Sets – C und JavaScript/HTML/CSS

Dieses Repository enthält meine Lösungen für die Problem Sets aus CS50X, umgesetzt in verschiedenen Programmiersprachen.

## Struktur

- `PSET1/` – Basics der Programmiersprache C: Datenstrukturen,Conditionals,Loops
- `PSET2/` – Verschiedene Aufgaben zum Vertiefen der Grundlagen in C
- `PSET3/` – Fortgeschrittene Aufgaben zur Auszählung von Wählerstimmen
- `PSET4/` - Erste statische Homepage mit HTML, CSS, JavaScript
- `PSET5/` - Aufgabe zur Erkennung von Mustern in großen Datenmengen

## Hinweise

- Jede Datei enthält meine eigene Implementierung und wurde ohne KI-Unterstützung bearbeitet.
- Die Lösungen sind funktional und zeigen mein Verständnis für die zugrundeliegenden Programmiersprachen und mein analytisches Denken.
